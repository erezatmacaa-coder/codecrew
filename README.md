# CodeCrew 🤖

**Multi-Agent Kod Geliştirme Sistemi** — 8 yapay zeka ajanı bir ofiste birlikte çalışır, kod yazar ve projeleri tamamlar.

## Özellikler

- 🧠 **8 AI Ajanı** — Her biri farklı uzmanlıkta (frontend, backend, test, dokümantasyon vb.)
- 🤝 **Otonom İş Birliği** — Ajanlar kendi aralarında iletişim kurar, görev dağıtır
- 📝 **Otomatik Kod Üretimi** — Proje şablonundan tam çalışan uygulamaya
- 🔄 **Git Entegrasyonu** — Otomatik commit ve push
- 🎯 **Proje Yönetimi** — Görev takibi, ilerleme raporu

## Kullanım

```bash
python main.py
```

## Gereksinimler

- Python 3.10+
- pip install -r requirements.txt
